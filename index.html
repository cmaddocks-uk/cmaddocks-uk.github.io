About

I'm Christopher Maddocks, a former ANME (Association of Network Managers in Education) Ambassador. These tools are built and maintained as free contributions to the UK education community — designed to be genuinely useful for schools, colleges and multi-academy trusts working towards the DfE Digital Standards 2030 and a stronger cyber posture.

All tools run entirely in your browser. No data is sent anywhere — no accounts, no cookies, no fingerprinting. Anonymous page-view counts via GoatCounter (privacy-friendly, GDPR-compliant) so I can see broadly how the tools are used.

Source code lives on GitHub. Found a bug, want to suggest an improvement, or want to say thanks? Each tool has its own feedback form and there's a Buy me a beer link if you've found these useful.
